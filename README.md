## Portfolio Website (React + TypeScript + Vite)

Developed a modern portfolio website using **React**, **TypeScript**, **Vite**, **Tailwind CSS**, and **shadcn-ui**.  
The project is hosted on **Lovable.dev** with automated deployment and version control integration through **GitHub**.
